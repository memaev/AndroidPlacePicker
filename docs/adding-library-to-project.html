Adding library to your project

In order to add library to your project you need to do the following:

  • Add Jitpack to your settings.gradle

  • Add dependency to your app level build.gradle

  • Add place picker activity to your AndroidManifest.xml

Adding jitpack

To your settings.gradle.kts add Jitpack import:

dependencyResolutionManagement { ... repositories { maven{ url = uri("https://jitpack.io") } } }
dependencyResolutionManagement { ... repositories { maven { url 'https://jitpack.io' } } }

Adding dependency to build.gradle

Add the following dependency to app module build.gradle file:

dependencies{ ... implementation("com.github.memaev:AndroidPlacePicker: [current-version]") }
dependencies{ ... implementation 'com.github.memaev:AndroidPlacePicker: [current-version]' }

Replace [current-version] with the latest version of the library. You can view it in releases tab.

Add place picker activity to AndroidManifest.xml

In order to open new activity for place picking you need to mark it in your manifest file. To do it go to manifest -> AndroidManifest.xml and add new activity to your application:

<application ... > <activity android:name="dem.llc.placepicker.presentation.activity.PlacePickerActivity" /> </application>
